## 🎮 About

**Vertex Studio** is an independent game development studio focused on creating **indie games and experimental gameplay experiences**.

The studio is currently in its early stage and is run by a solo developer passionate about learning and building games from the ground up.

Although no games have been officially released yet, development is focused on studying, prototyping ideas, and building the foundation for future projects.

The long-term goal is to create **immersive 3D indie games**, while also developing **smaller 2D projects** to gain experience and release games along the way.
